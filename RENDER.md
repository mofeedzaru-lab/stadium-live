علىعلى# دليل رفع تطبيق "إستاد تي في" على منصة Render 🚀

تم تجهيز المشروع بالكامل ليكون جاهزاً للرفع الفوري على منصة **Render** بأعلى كفاءة وأداء، ويدعم المشروع كلا الطريقتين المتاحتين على المنصة:

---

## 💡 الخيار الأول: الرفع كـ Static Site (مستحسن ومجاني بالكامل)
بما أن التطبيق عبارة عن تطبيق React (SPA) يتصل بـ Firebase مباشرة من طرف العميل (Client-Side)، فإن رفعه كموقع سكوني **Static Site** هو الخيار الأفضل لأنه مجاني كلياً وسريع الاستجابة ولا يتأثر بوقت إيقاف الخوادم (Cold Starts).

### خطوت الإعداد على Render:
1. قم بإنشاء حساب على [Render](https://render.com) واربطه بحساب GitHub الخاص بك.
2. اضغط على **New +** ثم اختر **Static Site**.
3. حدد مستودع المشروع الخاص بك (GitHub Repository).
4. اضبط الإعدادات التالية في نموذج الإنشاء:
   * **Name**: `staad-tv` (أو أي اسم تفضله).
   * **Build Command**: `npm install && npm run build`
   * **Publish Directory**: `dist`
5. اضغط على **Create Static Site** وبذلك يتم بناؤه وتشغيله فوراً!

> **ملاحظة هامة (اختيارية):** إذا واجهت مشاكل في المسارات البديلة (Routing) عند التصفح الداخلي، يمكنك إضافة قاعدة إعادة توجيه (Redirect Rules) في لوحة تحكم Render الخاصة بالموقع السكوني:
> * **Source**: `/*`
> * **Destination**: `/index.html`
> * **Status**: `200 (Rewrite)`

---

## 🖥️ الخيار الثاني: الرفع كـ Web Service (باستخدام Node.js)
إذا كنت تفضل تشغيل خادم خلفي كامل يدير عملية توجيه حركة المرور وتقديم الملفات مسبقة الصنع:

تمت إضافة خادم ويب Express خفيف وسريع (`server.js`) وإدراج أمر البدء التلقائي في ملف الإعدادات `package.json`.

### خطوات الإعداد على Render:
1. اضغط على **New +** ثم اختر **Web Service**.
2. حدد مستودع المشروع الخاص بك من GitHub.
3. اضبط خيارات البيئة كالتالي:
   * **Runtime**: `Node`
   * **Build Command**: `npm install && npm run build`
   * **Start Command**: `npm start`
4. اضغط على **Create Web Service**، وسيقوم Render ببناء الواجهات وتشغيل الخادم فوراً على المنفذ المطلوب.

---

## 🔒 إعداد المتغيرات الساقطة وقاعدة البيانات (Firebase Config)
إذا قمت بربط التطبيق بقاعدة بيانات Firebase، فتأكد من تمرير متغيرات البيئة الخاصة بك في لوحة تحكم Render ضمن تبويب **Environment** لضمان عمل قاعدة البيانات والمزامنة بكفاءة عالية وأمان تام.
